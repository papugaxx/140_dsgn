Оформить заказ

Форма в полной Django-версии сохраняет заявку, привязывает её к пользователю и принимает файлы.

Техническое задание

Услуга: Баннер
Название: Баннер для Telegram-канала
Описание: стиль, размеры, текст, референсы и пожелания...
Контакт: @username
Материалы: brief.pdf, references.zip
Отправить заказ

Что происходит дальше

1. Клиент отправляет ТЗ
2. Дизайнер уточняет детали
3. Появляются цена и дедлайн
4. Финальный файл доступен в кабинете

На GitHub Pages показана статическая версия интерфейса. Рабочая форма доступна в Django-проекте при локальном запуске.
