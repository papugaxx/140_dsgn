# Design Flow dense grid polish

Финальная полировка после скриншотов: уменьшена высота футера, выровнены отступы, исправлено налезание прогресса заказа, карточки услуг/заказов стали менее зажатыми, текст и кнопки не конфликтуют. Контентная dense-сетка сохранена. CSS подключается через static, inline-стилей в Django-шаблонах нет.
